# jquery_circle_maker
jquery plugin practice
 
 *  参数说明 
 *  target:圆环包裹元素(Dom)
 *  strokeStyle:圆边框颜色(String)
 *  strokeWidth:圆边框大小(Number)
 *  circleWidth:圆的大小(Number)
 *  circleTiem:画圆的时间(Number毫秒)
 ##使用方法
```javascript
  $(target).MakeCircle({
    strokeStyle:"#0f0",
    strokeWidth:50,
    circleWidth:500,
    circleTime:2500
  });
```
